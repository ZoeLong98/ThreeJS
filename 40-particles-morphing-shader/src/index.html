Project Summary

  • 1. Particle Text: The text models are created in Blender, and their vertex positions are exported. THREE.Points is used to generate the particles in Three.js.
  • 2. Particle Morphing: The morphing animation is achieved using shaders to interpolate between different particle states. Based on a noise function, vertices start and end at different times, which is also used on vertex colors.
  • 3. Camera Control: The camera behavior is implemented in JS by recording the current and next model indices, morphing progress, model centers, and model normals.

This project is based on the Three.js Journey Particles Morphing Shader course
